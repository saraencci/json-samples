Privacy Policy

Jumia Cross-Selling Tool Chrome Extension

This Privacy Policy explains how the Cross-Selling Tool Chrome Extension handles user information.

1. Information Collection

The Cross-Selling Tool does not collect personally identifiable information such as name, email, phone number, or passwords.

However, the extension collects limited non-personal interaction data for analytics and performance improvement purposes.

2. Data We Collect

The extension may collect the following non-personal usage data:

  • Current page URL
  • Product or item name displayed on the page
  • Recommended product clicked
  • Destination URL of clicked recommendation
  • Timestamp of interaction

Example data:

{
  "item_url": "current page URL",
  "next_page_url": "clicked product URL",
  "item_clicked": "product name",
  "created_at": "timestamp"
}
This data is used strictly for analytics such as measuring recommendation performance and improving relevance.

3. How Data Is Used

Collected data is used for:

  • Improving product recommendation relevance
  • Measuring click-through performance (CTR)
  • Debugging and improving extension functionality
  • Aggregated analytics and performance insights

4. Data Storage

Collected data is securely stored in a Supabase database under a usage analytics table.

No personal identity data is stored, and all records are limited to interaction events.

5. Third-Party Services

This extension uses Supabase for secure storage of anonymized interaction events.

No advertising networks, trackers, or unrelated third-party analytics tools are used.

6. Data Access & Security

Only insert operations are allowed from the extension. No sensitive user data is accessible or exposed.

7. Data Retention

Data may be retained for analytics purposes and may be periodically cleaned or aggregated. No personal profiling is performed.

8. Permissions

The extension requires access to supported e-commerce websites in order to:

  • Display product recommendations
  • Track interaction events with recommendations

No browsing history outside supported pages is accessed.

9. Changes to This Policy

This privacy policy may be updated if the extension functionality changes. Updates will be reflected in this document.
